สร้างปุ่มโต้ตอบด้วย CSS

รายละเอียดงาน

คุณได้รับมอบหมายให้สร้างปุ่ม HTML พื้นฐานและใช้ CSS เพื่อทำให้มันดูน่าสนใจและตอบสนองเมื่อผู้ใช้โต้ตอบด้วย (เช่น เมื่อนำเมาส์ไปชี้) เป้าหมายคือการฝึกฝนการใช้งานคุณสมบัติ CSS พื้นฐานและการจัดการสถานะ hover

สิ่งที่โจทย์ต้องการ

  • เปลี่ยนสีพื้นหลังของปุ่มเป็นสีฟ้า (#007bff)
  • เปลี่ยนสีตัวอักษรเป็นสีขาว (#ffffff)
  • เพิ่มระยะห่างภายใน (padding) ด้านบนและล่าง 10px, ด้านซ้ายและขวา 20px
  • ทำให้มุมของปุ่มมนเล็กน้อย (border-radius) 5px
  • ลบเส้นขอบเริ่มต้นของปุ่มออก (border: none;)
  • เปลี่ยนรูปแบบเคอร์เซอร์เป็น "pointer" เมื่อนำเมาส์ไปชี้ที่ปุ่ม (cursor: pointer;)
  • เมื่อนำเมาส์ไปชี้ที่ปุ่ม (hover), ให้เปลี่ยนสีพื้นหลังเป็นสีน้ำเงินเข้มขึ้น (#0056b3)
