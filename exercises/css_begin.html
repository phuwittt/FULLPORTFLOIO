สร้างการ์ดสินค้าอย่างง่ายด้วย CSS

รายละเอียดงาน

คุณได้รับมอบหมายให้สร้างการ์ดสินค้า (product card) โดยใช้ HTML และ CSS เพื่อแสดงข้อมูลสินค้าพื้นฐาน การ์ดควรมีรูปภาพสินค้า ชื่อสินค้า ราคา และปุ่ม 'เพิ่มลงตะกร้า' โจทย์นี้จะช่วยให้คุณฝึกฝนการใช้งาน CSS พื้นฐาน เช่น การจัดองค์ประกอบ สี และการจัดวาง

สิ่งที่โจทย์ต้องการ

  • ทำให้การ์ดสินค้า (`.product-card`) มีความกว้าง 300px และมีขอบมนเล็กน้อย (border-radius) ที่ 8px
  • เพิ่มเงาให้กับการ์ด (`.product-card`) เพื่อให้ดูมีมิติมากขึ้น
  • จัดตำแหน่งข้อความและปุ่มทั้งหมดภายใน `.product-card` ให้อยู่กึ่งกลาง
  • ตั้งค่าขนาดรูปภาพสินค้า (`.product-image`) ให้มีความกว้าง 100% ของการ์ดและมีความสูงคงที่ 200px (ใช้ object-fit: cover เพื่อให้รูปภาพไม่บิดเบี้ยว)
  • เปลี่ยนสีพื้นหลังของปุ่ม 'เพิ่มลงตะกร้า' (`.add-to-cart-btn`) เป็นสีเขียว และเปลี่ยนสีตัวอักษรเป็นสีขาว
  • เพิ่ม padding รอบข้อความและปุ่มภายใน `.product-card` เพื่อให้มีช่องว่าง
