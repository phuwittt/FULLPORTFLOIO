สร้างระบบแสดงสินค้าแบบ Grid ที่ตอบสนอง

รายละเอียดงาน

คุณได้รับมอบหมายให้สร้างหน้าเว็บที่แสดงรายการสินค้าหลายชิ้นในรูปแบบตาราง (grid) ที่สามารถปรับเปลี่ยนการแสดงผลได้ตามขนาดหน้าจอ (responsive) เพื่อให้ผู้ใช้งานได้รับประสบการณ์ที่ดีที่สุดไม่ว่าจะเข้าชมจากอุปกรณ์ใดๆ

สิ่งที่โจทย์ต้องการ

  • สร้างโครงสร้าง HTML สำหรับการ์ดสินค้าอย่างน้อย 6 ชิ้น โดยแต่ละการ์ดควรมีรูปภาพ ชื่อสินค้า และราคา
  • จัดเรียงการ์ดสินค้าให้อยู่ในรูปแบบตาราง (grid) โดยใช้ CSS Flexbox หรือ CSS Grid
  • ทำให้หน้าเว็บตอบสนอง (responsive) โดยที่: บนหน้าจอขนาดเล็ก (เช่น มือถือ) การ์ดควรแสดง 1 คอลัมน์, บนหน้าจอขนาดกลาง (เช่น แท็บเล็ต) การ์ดควรแสดง 2 คอลัมน์, บนหน้าจอขนาดใหญ่ (เช่น คอมพิวเตอร์) การ์ดควรแสดง 3 หรือ 4 คอลัมน์
  • เพิ่มสไตล์พื้นฐานให้กับการ์ดสินค้า เช่น ขอบเงา (box-shadow) ระยะห่าง (padding/margin) และการจัดตำแหน่งข้อความ
  • รูปภาพสินค้าควรปรับขนาดให้เหมาะสมกับการ์ดและไม่ยื่นออกนอกกรอบ
