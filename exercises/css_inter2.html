สร้างตารางการ์ดที่ตอบสนอง (Responsive Card Grid) พร้อมเอฟเฟกต์เมื่อวางเมาส์

รายละเอียดงาน

โจทย์นี้จะให้คุณสร้างหน้าเว็บที่มีชุดการ์ดสินค้าหรือบริการ จัดเรียงในรูปแบบตาราง (Grid) ที่สามารถปรับขนาดได้ตามหน้าจอ (Responsive) การ์ดแต่ละใบควรมีรูปภาพ, ชื่อ, และคำอธิบายสั้นๆ นอกจากนี้ เมื่อผู้ใช้วางเมาส์เหนือการ์ดแต่ละใบ ควรมีเอฟเฟกต์การเปลี่ยนภาพที่น่าสนใจและราบรื่น

สิ่งที่โจทย์ต้องการ

  • ใช้ CSS Grid ในการจัดเรียงการ์ดภายใน `.container` ให้เป็นตาราง
  • การ์ดต้องตอบสนองต่อขนาดหน้าจอโดยใช้ Media Queries:
  • แสดง 1 คอลัมน์บนหน้าจอขนาดเล็ก (เช่น `max-width: 600px`)
  • แสดง 2 คอลัมน์บนหน้าจอขนาดกลาง (เช่น `min-width: 601px` และ `max-width: 900px`)
  • แสดง 3 คอลัมน์บนหน้าจอขนาดใหญ่ (เช่น `min-width: 901px`)
  • เพิ่มเอฟเฟกต์เมื่อวางเมาส์ (hover effect) บน `.card` เช่น การ์ดขยายเล็กน้อย (`transform: scale(1.03);`) และ/หรือมีเงาเพิ่มขึ้น (`box-shadow: 0 8px 25px rgba(0, 0, 0, 0.2);`)
  • ตรวจสอบให้แน่ใจว่าเอฟเฟกต์การวางเมาส์เป็นไปอย่างราบรื่นโดยใช้ CSS Transitions
