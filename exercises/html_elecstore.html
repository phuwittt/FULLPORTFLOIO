สร้างหน้าแสดงสินค้าอิเล็กทรอนิกส์

รายละเอียดงาน

คุณได้รับมอบหมายให้สร้างหน้าเว็บ HTML สำหรับแสดงรายการสินค้าอิเล็กทรอนิกส์ หน้าเว็บควรมีการจัดเรียงข้อมูลสินค้าอย่างเป็นระเบียบ ใช้แท็ก HTML ที่เหมาะสมเพื่อสื่อความหมายของเนื้อหา และเตรียมโครงสร้างพื้นฐานสำหรับการเพิ่มสไตล์ CSS ในอนาคต

สิ่งที่โจทย์ต้องการ

  • ใช้แท็ก HTML5 เชิงความหมาย (semantic HTML5 tags) เช่น header, nav, main, article, section, footer ให้เหมาะสม.
  • มีส่วนหัว (header) ที่ประกอบด้วยโลโก้ (text-based logo หรือ img) และแถบเมนูนำทาง (navigation menu) อย่างน้อย 3 ลิงก์.
  • แสดงรายการสินค้าอย่างน้อย 3 ชิ้นภายในแท็ก main โดยแต่ละชิ้นควรมี: รูปภาพสินค้า, ชื่อสินค้า, คำอธิบายสั้นๆ, ราคา และปุ่ม 'เพิ่มลงตะกร้า'.
  • ใช้แท็ก ul หรือ ol สำหรับการจัดกลุ่มรายการสินค้า.
  • ใช้แท็ก img สำหรับรูปภาพสินค้า พร้อมคุณสมบัติ alt ที่เหมาะสม.
  • มีส่วนท้าย (footer) ที่แสดงข้อมูลลิขสิทธิ์และลิงก์ติดต่อ.
