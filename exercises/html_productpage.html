รายละเอียดงาน

สร้างหน้าเว็บ HTML สำหรับแสดงรายละเอียดของสินค้า โดยใช้โครงสร้าง HTML เชิงความหมาย (semantic HTML) ที่เหมาะสม เพื่อให้เข้าถึงได้ง่ายและมีโครงสร้างที่ดีสำหรับ SEO และการประมวลผลด้วยเครื่องมือต่างๆ หน้าเว็บควรประกอบด้วยข้อมูลสินค้า รูปภาพ แผนภูมิราคา และส่วนแสดงความคิดเห็น/รีวิวสินค้า

สิ่งที่โจทย์ต้องการ

  • ใช้แท็ก header, nav, main, section, article, aside, footer ให้เหมาะสมกับโครงสร้างของหน้า
  • แสดงชื่อสินค้าและคำอธิบายโดยใช้แท็กหัวข้อ (h1, h2) และย่อหน้า (p)
  • มีส่วนแสดงรูปภาพสินค้าหลักอย่างน้อย 1 รูป และรูปภาพขนาดย่อ (thumbnail) อย่างน้อย 3 รูป โดยใช้แท็ก img และมี alt attribute ที่เหมาะสม
  • แสดงข้อมูลราคา (ราคาปัจจุบัน, ราคาลด) โดยใช้แท็กที่เหมาะสม เช่น span หรือ del สำหรับราคาเก่า
  • สร้างตาราง HTML (table) เพื่อแสดงคุณสมบัติสินค้า (เช่น ขนาด, สี, วัสดุ) โดยมี thead, tbody และอย่างน้อย 3 แถวข้อมูล
  • มีฟอร์มสำหรับเขียนรีวิวสินค้า (form) ซึ่งประกอบด้วย: ช่องใส่ชื่อผู้รีวิว (input type="text"), ช่องให้คะแนนสินค้า (เช่น select หรือ input type="range"), ช่องสำหรับข้อความรีวิว (textarea), ปุ่มส่งรีวิว (input type="submit"), ใช้ label คู่กับ input และ textarea, ใช้ fieldset และ legend สำหรับจัดกลุ่มฟอร์ม
  • มีส่วนแสดงความคิดเห็น/รีวิวที่มีอยู่แล้วอย่างน้อย 2 รายการ โดยแต่ละรีวิวควรใช้แท็ก article ภายใน section และแสดงชื่อผู้รีวิว วันที่ และข้อความรีวิว
  • มีส่วน "สินค้าที่เกี่ยวข้อง" (aside) ที่แสดงลิงก์ไปยังสินค้าอื่น ๆ อย่างน้อย 3 รายการ โดยใช้ ul และ li
